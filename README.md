# 站点状态监控
- 本项目可以直接部署，快速上线，纯静态无需服务器。基于 UptimeRobot api 时时监控主机、网站、端口等在线状态；

## 在线体验：  

![](https://gcore.jsdelivr.net/gh/Qikaile/cdn/img/Snipaste_2022-08-09_09-25-53.jpg)


### 安装依赖
```
npm i --unsafe-perm=true
```
### 运行
```
npm run start
```
### 打包
```
npm run build
```
